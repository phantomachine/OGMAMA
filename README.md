# OGMAMA
**O**pen-source, transparent **G**rading, **M**anagement of **A**ssessments, and **M**onitoring of **A**ctive-learning repository.

* `Jupyter` notebooks
* Uses mostly `Pandas` and `NumPy` tools

The raw 0/1-mark results from multiple choice question (MCQ) sections of exams/assessments are re-processed to allow for possible re-weighting of questions. That is, examiners may set MCQs with different marks/weights. THe tool to do this processing can be found here:

* [MCQ-final](https://github.com/phantomachine/MCQ-final)
